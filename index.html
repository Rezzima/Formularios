¿Qué atributo se utiliza para especificar la URL a la que se enviarán los datos del formulario? a) method b) action c) target d) url

Se usa la opcion "b"

¿Cuál es el atributo que se utiliza para especificar el tipo de datos que se espera en un campo de entrada? a) data-type b) input-type c) type d) data

Se usa el atributo "type"

¿Cuál de los siguientes tipos de campo de entrada se utiliza para ingresar direcciones de correo electrónico? a) text b) email c) mail d) input

Se usa "email"

¿Cuál es el elemento HTML que se utiliza para asociar un texto descriptivo con un campo de entrada en un formulario? a) b) c) d)

Se usa "label"

¿Cuál de las siguientes opciones es la etiqueta correcta para crear un botón de envío en un formulario? a) "input type="submit" b) "button type="submit" c) "btn" d) "submit>/submit"

Se usa "input type="submit"

¿Cuál de los siguientes atributos se utiliza para hacer que un campo de entrada sea obligatorio en un formulario? a) required b) mandatory c) validate d) mandatoryfield

Se usa "required"

¿Cuál de los siguientes elementos HTML se utiliza para crear una lista desplegable en un formulario? a) "list" b) "dropdown" c) "select" d) "input type="list">

Se usa "select"

¿Cuál de los siguientes tipos de campo de entrada se utiliza para ingresar un apellido? a) text b) email c) mail d) input

Se usa "text"

¿Qué atributo se utiliza para definir un valor predeterminado en un campo de entrada de un formulario? a) value b) default c) preset d) initial

Se usa "default"
